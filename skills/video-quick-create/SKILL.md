# video-quick-create

> **注意**：如果是首次使用，确保已安装 videoclaw：`uvx videoclaw --help`

## 触发场景

用户说"做一个xx视频"、"帮我生成一个视频"时触发。

## 输入

用户需要提供：
- 视频主题/想法
- 角色（如有）
- 时长偏好

## 执行流程

### 0. 确认资产图来源

用 AskUserQuestion 询问：
> "你有人物九宫格资产图吗？"

- 有 → 跳过步骤 1，直接用用户图片
- 没有 → 继续步骤 1

如无资产，追问：
> "有人物参考图吗？"

- 有 → I2I 模式
- 没有 → T2I 模式

### 1. 生成九宫格资产

**九宫格角色图内容**（3x3网格，共9个位置）：
1. 正面全身照
2. 侧面全身照（左侧）
3. 背面全身照
4. 3/4 侧面照（左右各一）
5. 面部特写
6. 表情变化（微笑、惊讶等）
7. 服装/装扮细节
8. 常用动作姿态（如行走、站立）
9. 上面/俯视角度

**T2I 模式**:

1. 根据角色描述构建 prompt：
```
九宫格角色图，3x3网格分布，包含：
- 位置1：正面全身照，角色描述
- 位置2：侧面全身照（左），角色描述
- 位置3：3/4侧面照（左），角色描述
- 位置4：背面全身照，角色描述
- 位置5：3/4侧面照（右），角色描述
- 位置6：侧面全身照（右），角色描述
- 位置7：面部特写，角色描述
- 位置8：表情变化，角色描述
- 位置9：服装细节/常用姿态，角色描述

要求：风格统一，光照一致，背景纯色, 适合作为视频生成的参考
```

2. 用 AskUserQuestion 询问用户：
> "这是生成的 prompt，满意吗？需要调整吗？"
> - 满意 → 继续生成
> - 调整 → 询问具体要改什么

3. 确认后执行：
```bash
videoclaw t2i -p "<构建好的prompt>" -o <project>/assets/character_grid.png
```

4. 生成后，用 AskUserQuestion 询问：
> "九宫格角色图已生成，满意吗？"
> - 满意 → 继续下一步
> - 调整 → 询问具体要调什么
> - 重新生成 → 不调整直接重试

**I2I 模式**:

1. 构建 prompt：
```
转换为九宫格角色图，3x3网格分布：
- 正面全身、侧面（左右）、背面、3/4侧面（左右）
- 面部特写、表情变化
- 服装细节、常用姿态

保持原图的角色特征和风格，统一光照，背景纯色
```

2. 用 AskUserQuestion 确认 prompt

3. 执行：
```bash
videoclaw i2i -i <reference.jpg> -p "<构建好的prompt>" -o <project>/assets/character_grid.png
```

4. 生成后，用 AskUserQuestion 询问：
> "九宫格角色图已生成，满意吗？"
> - 满意 → 继续下一步
> - 调整 → 询问具体要调什么
> - 重新生成 → 不调整直接重试

### 2. 生成文本分镜

调用 video-text-storyboard skill 生成结构化文本分镜。

### 3. 生成视频

从第二步获取文本分镜内容，传入 -t 参数：

```bash
videoclaw i2v -p <project> -i <assets/image1.png> -i <assets/image2.png> -t "<第二步生成的文本分镜内容>"
```

## 交互确认

每步完成后用 AskUserQuestion 询问：
- "满意吗？" → 满意 / 调整 / 重新生成
