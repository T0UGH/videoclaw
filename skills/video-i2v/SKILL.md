# video-i2v - 图生视频

## 概述
将故事板图片转换为视频片段。

## 使用方式

```bash
videoclaw i2v --project my-project
```

## 参数

- `--project, -p`: 项目名称（必填）

## 前置条件

需要先完成 `video:storyboard`

## 示例

```
Claude Code: videoclaw i2v --project mars-video
```
