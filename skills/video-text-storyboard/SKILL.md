# video-text-storyboard

> **注意**：如果是首次使用，确保已安装 videoclaw：`uvx videoclaw --help`

## 触发场景

用户说以下话时触发：
- "帮我生成一个分镜"
- "帮我写视频脚本"
- "帮我设计一下这个视频怎么拍"
- "帮我做一个视频方案"

## 输入

通过 AskUserQuestion 追问获取：
1. **视频主题**："这个视频想讲什么？"
2. **角色**："有角色吗？如果有，描述一下角色外观"
3. **时长**："视频需要多长？15秒/30秒/1分钟"
4. **风格**："想要什么风格？搞笑/温馨/史诗感/其他"

## 生成要点

加载 skill 后，利用上下文生成文本分镜。

### 模板选择

根据视频类型选择对应模板，详见 [references/stories.md](references/stories.md)：

- **故事类**：使用故事类模板（见 references/stories.md）

### 故事类模板结构

```
【素材介绍】：
【@图1】<角色/物体描述>
【@图2】<角色/物体描述>

【场景描写】：
<整体背景描述>

【背景音】：
<背景音描述>

【分镜】：
【镜头标题】（X-Y秒）
画面：<包含动作、台词、情绪的完整描述>
音效：<此镜头的音效>

【注意事项】：
- 角色设定：<关键角色特征>
- 氛围：<整体氛围要求>
- 动作节奏：<节奏要求>
- 细节还原：<关键细节>
```

### 分镜生成要点

每个分镜包含：
- **时间范围**：精确的秒数（如 2-6秒）
- **画面**：场景 + 动作 + 台词 + 情绪，融合在一个字段中
- **音效**：此镜头的背景音效

### 合格示例

**输入**：做一个冰霜王登基的动画视频

**输出**：

```
【素材介绍】：
【@图1】黄色圆滚滚小怪兽，无鼻设计，头顶小橘子，眼神坚定，气场凛冽
【@图2】冰封圣剑，剑身蓝光流转，插在寒冰基座上
【@图3】冰甲去头盔版本，覆盖全身的完整冰晶铠甲
【@图4】穿戴冰甲的小怪兽形态
【@图5】冰之王冠，由冰晶碎片拼接而成
【@图6】冰王形态的小怪兽，手持冰封圣剑
【@图7】冰封王座，台阶和座椅有冰晶纹路

【场景描写】：
极寒冰封之地，浮空冰峰刺破阴云，寒风呼啸，冰封大殿寂静威严

【背景音】：
史诗感 夹杂冰面细微碎裂声

【分镜】：
【拔剑】（2-6秒）
画面：冰封圣剑【@图2】特写 → 黄色小怪兽【@图1】握爪手部特写。冰封圣剑插在寒冰基座上，蓝光流转；黄色小怪兽肉垫爪子紧紧握住剑柄，头顶小橘子微微颤动，眼神坚定，气场凛冽
音效：冰裂声、低沉嗡鸣

【冰甲飞覆】（6-10秒）
画面：拔剑动作中景 → 环绕运镜跟随冰晶。黄色小怪兽【@图1】奋力拔剑，圣剑【@图2】爆发刺眼蓝光；无数冰晶碎片从四面八方飞聚而来，瞬间拼接成完整冰甲去头盔版本【@图3】，覆盖全身【@图4】
音效：剑鸣、冰晶碰撞声、寒冰碎裂声

【自戴王冠】（10-14秒）
画面：冰片汇聚慢镜头 → 小怪兽抬手特写。无数冰晶冰片从空中飞聚而来，在小怪兽【@图4】面前缓缓拼成一顶完整冰之王冠【@图5】；小怪兽抬手，将王冠【@图5】戴在自己头上，双眼亮起冰蓝光芒，披风展开
音效：冰晶拼接声、寒风呼啸

【王座宣言】（14-18秒）
画面：台阶推进镜头 → 王座全景仰拍。黄色小怪兽（冰王形态）【@图6】手持冰封圣剑【@图2】，一步步踏上冰封王座台阶，每一步冰面震颤，王座【@图7】纹路发光
音效：沉稳脚步声、王座共鸣

【终章定格】（18-20秒）
画面：王冠【@图5】小橘子特写 → 冰峰远景拉镜。特写王冠上的冰晶小橘子，黄色小怪兽眼神冷冽霸气；远处冰峰堡垒在背景中矗立
音效：余音回荡

【注意事项】：
- 角色设定：黄色小怪兽严格保持无鼻设计，保证角色辨识度
- 氛围：全程冷色调，突出寒冰、冰晶、冰甲质感，与角色暖黄色形成强烈视觉对比
- 动作节奏：拔剑 → 冰甲飞覆 → 自戴王冠 → 登王座一气呵成，卡点音效，保证20秒高燃节奏
- 细节还原：头顶小橘子、冰封圣剑蓝光、冰甲拼接纹路等关键细节必须清晰呈现
```

## 保存

生成后保存到 `<project>/storyboard/text_storyboard.md`

## 交互确认

用 AskUserQuestion 询问用户：
> "这是生成的分镜，满意吗？"

选项：
- 满意 → 继续下一步
- 修改 → 询问具体哪里要改
- 重新生成 → 不修改直接重试

## 常见错误

❌ 每个镜头安排太多内容：视频生成模型4秒绝对执行不了这么多东西，一个镜头只安排一个简单动作

✅ 每个镜头内容要精简，只描述一个动作或变化

---

❌ 素材引用混乱：不说明引用了什么

✅ 使用【@图1】【图2】格式明确引用素材

---

❌ 忽视音效：音效设计能大幅提升质量

✅ 加入音效：背景音乐 + 音效

---

❌ 缺少注意事项：结尾没有强调关键要素

✅ 添加注意事项：角色设定、氛围、动作节奏、细节还原

❌ 镜头数量太少：1-2个镜头无法展现完整故事

✅ 多用镜头没问题，每个镜头内容细化，但总量控制在30秒左右（6-8个镜头）

---

❌ 段落之间有多余空行：每个段落、镜头之间不需要空行

✅ 全部连续书写，不要有空行
