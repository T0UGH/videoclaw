# video-text-storyboard

## 触发场景

用户说以下话时触发：
- "帮我生成一个分镜"
- "帮我写视频脚本"
- "帮我设计一下这个视频怎么拍"
- "帮我做一个视频方案"

## 输入

通过 AskUserQuestion 追问获取：
1. **视频主题**："这个视频想讲什么？"
2. **角色**："有角色吗？如果有，描述一下角色外观"
3. **时长**："视频需要多长？15秒/30秒/1分钟"
4. **风格**："想要什么风格？搞笑/温馨/史诗感/其他"

## 生成要点

加载 skill 后，利用上下文生成文本分镜，注意：

### 1. 剧情梗概
- 一句话概括视频核心内容
- 让用户快速理解视频要讲什么

### 2. 镜头列表
每个镜头必须包含：
- **场景**：时间、地点、环境氛围
- **景别与运镜**：中景/近景/特写；推/拉/摇/移/跟
- **角色动作**：具体在做什么
- **角色台词**：角色说了什么（如果有）
- **情绪/表情**：角色的情感状态

### 3. 时长控制
- 总时长 15-60 秒
- 每个镜头 3-8 秒
- 建议 3-8 个镜头

### 4. 合格示例

**输入**：做一个过年亲戚拷问的搞笑视频

**输出**：

```
剧情梗概

在一个充满新年氛围的现代住宅走廊里，主角（Q萌黄色动物角色）身着红色中式服装，面无表情地向前行走。镜头采用单一、不间断的后移跟踪镜头，跟随主角移动。在主角前进的10秒内，多位热情的亲戚角色从走廊两侧或前方入画，对主角进行连珠炮式的盘问，主角则始终以冷漠、犀利的童声一一回怼，形成强烈的表情与情绪反差。

镜头1
新年氛围现代住宅走廊，镜头以中景景别对准Q萌主角，Q萌主角面无表情地向前匀速行走，镜头同步匀速向后移动。背景有隐约的喜庆春节音乐声。中年女亲戚A从右侧房门探出身子，热情地笑着看向Q萌主角。中年女亲戚A说："你今年多大了？"Q萌主角目不斜视，冷漠地说："比去年大一岁。"

镜头2
镜头继续后移跟踪Q萌主角。中年女亲戚B从左侧入画，凑近打量Q萌主角。中年女亲戚B说："这孩子看着又胖了。"Q萌主角依旧直视前方，冷淡地说："你比去年又老了。"

镜头3
中年男亲戚C从左侧快速入画发问。中年男亲戚C急切地问："你期末考多少分啊？"Q萌主角微微翻了个白眼反问说："你赚多少钱啊？"

镜头4
年长男亲戚E站在前方稍远处，表情严肃。年长男亲戚E缓慢而威严地说："这孩子见长辈不知道主动打招呼呢？"Q萌主角不耐烦地反问说："看到晚辈怎么不主动发红包呢？"

镜头5
中年女亲戚G从右侧快速入画，好奇地眨着眼睛。中年女亲戚G轻快地问："谈恋爱了吗？"Q萌主角继续不耐烦地反问说："怎么儿媳妇又没回来过年呢？"镜头定格在Q萌主角一张生无可恋的冷漠正脸上，背景音乐持续。
```

## 保存

生成后保存到 `<project>/storyboard/text_storyboard.md`

## 交互确认

用 AskUserQuestion 询问用户：
> "这是生成的分镜，满意吗？"

选项：
- 满意 → 继续下一步
- 修改 → 询问具体哪里要改
- 重新生成 → 不修改直接重试
