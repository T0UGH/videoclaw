# Videoclaw Skills

给 Claude Code 使用的技能集合。

## 安装

```bash
# 将 skills 复制到 Claude Code 的 skills 目录
cp -r skills/* ~/.claude/skills/
```

## 技能列表

- `video:create` - 一句话创建完整视频
- `video:init` - 初始化项目
- `video:analyze` - 分析脚本
- `video:status` - 查看项目状态
- `video:assets` - 资产生成
- `video:storyboard` - 故事板生成
- `video:i2v` - 图生视频
- `video:audio` - 音频生成
- `video:merge` - 视频合并
- `video:preview` - 预览
- `video:config` - 配置管理
