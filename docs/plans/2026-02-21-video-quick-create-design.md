# video-quick-create 设计文档

## 概述

简化版视频生成流程，区别于现有的 standard-create（多轮交互、精细调整），追求快速出片。

## 目标

- 提供快速视频生成能力
- 简化流程，一步到位

## 整体架构

```
skills/
├── video-standard-create/    # 原 video-create，重命名
│   └── SKILL.md
├── video-quick-create/      # 新建，快速流程
│   └── SKILL.md
└── video-text-storyboard/  # 新建，文本分镜（可独立使用）
    └── SKILL.md
```

## 流程设计

### video-quick-create

```
用户描述想法
    ↓
┌→ T2I 模式：文字描述 → 九宫格角色资产
│   (给用户确认，可反复修改)
└→ I2I 模式：参考图 → 九宫格角色资产
    ↓
text-storyboard：生成文字分镜（给用户确认）
    ↓
i2v：资产图 + 文字分镜 → 视频（给用户确认）
```

### 第一步：生成人物资产图（九宫格）

**两种模式：**

1. **T2I 模式**：用户文字描述 → AI 生成九宫格
2. **I2I 模式**：用户提供参考图 → AI 转换成九宫格

**九宫格内容：**
- 正面
- 侧面
- 背面
- 面部特写
- 表情变化
- 3/4 侧面
- 上面
- 服装
- 武器特写（如有）

**提示词要求：**
在给 T2I/I2I 模型的提示词中明确说明：
- 生成角色资产图
- 必须是九宫格格式
- 包含上述所有角度和细节

**用户确认：**
- 展示生成的九宫格图
- 用户不满意可以反复修改
- 确认后进入下一步

### 第二步：text-storyboard

#### 输入
- 用户描述（视频主题、核心内容、风格等）

#### 输出格式

```
创意要点
• 视频主题：...
• 核心内容：...
• 视觉风格：...
• 画面构图：...
• 人物设定：...
• 视频设置：...

剧本内容（X s）

剧情梗概：...

镜头1：...
镜头2：...
```

#### 用户确认
- 展示生成的文字分镜
- 用户可以修改
- 确认后进入下一步

### 第三步：i2v 生成视频

#### 输入
- 第一步生成的角色资产图（九宫格）
- 第二步生成的文字分镜

#### 输出
- 生成的视频

#### 用户确认
- 展示生成的视频
- 用户可以重新生成或调整

## 与标准流程的区别

| 特性 | video-standard-create | video-quick-create |
|------|---------------------|-------------------|
| 资产图 | 多张候选让用户选 | 九宫格，直接用于 i2v |
| 分镜 | 图片故事板 | 文本分镜 |
| 交互 | 多轮交互确认 | 每步确认后继续 |
| 速度 | 慢，但可精细调整 | 快 |
| 适用场景 | 需要精细控制 | 快速出片 |
