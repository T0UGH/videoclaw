# video-quick-create 设计文档

## 概述

创建简化版视频生成流程：用户描述想法 → AI 生成文本分镜 → 图片 → 视频

## 目标

- 提供快速视频生成能力，区别于现有的 standard-create（多轮交互、精细调整）
- 简化流程，一步到位

## 整体架构

```
skills/
├── video-standard-create/    # 原 video-create，重命名
│   └── SKILL.md
├── video-quick-create/      # 新建，快速流程
│   └── SKILL.md
└── video-text-storyboard/   # 新建，文本分镜
    └── SKILL.md
```

## 流程设计

### video-quick-create

```
1. 用户描述视频想法
2. 调用 text-storyboard 生成文本分镜
3. 根据文本分镜生成图片
4. 调用火山 i2v 生成视频
```

### video-text-storyboard

#### 输入
- 用户描述（视频主题、核心内容、风格等）

#### 输出格式

```
创意要点
• 视频主题：Q版小怪兽与精英运动员的跨次元篮球对决
• 核心内容：黄色圆润、头顶小橘子、身着蓝橙球衣的Q版小怪兽球员，用灵活运球晃倒身穿23号紫色球衣的精英运动员后，从容完成三分跳投
• 视觉风格：超高画质，Q版萌系角色与写实精英运动员的跨次元对决感，充满幽默与动感
• 画面构图：画面中心，低角度仰拍，背景是灯光璀璨的职业篮球场，木地板反射顶光，带有细腻的动态模糊
• 人物设定：Q版小怪兽球员（可爱萌系）vs 精英运动员（平头黑人络腮胡，被晃倒在地，脸上写满不可思议的震惊）
• 视频设置：画面比例4:3，时长系统推荐，语言中文，显示字幕

剧本内容（13s）

剧情梗概：
在灯光璀璨的职业篮球场上，一场跨次元对决即将上演。Q版小怪兽球员以其矮胖可爱的萌系外形，对阵身形高大、肌肉线条分明的精英运动员。对决开始，小怪兽展现出与体型不符的惊人球技...

镜头1：
职业篮球场，低角度仰拍。画面中，精英运动员张开双臂...

镜头2：
镜头保持低角度仰拍，快速向左横摇...

镜头3：
镜头切换至更低角度的特写，带有动态模糊...

镜头4：
镜头迅速拉远并仰拍...
```

#### 实现方式

- 使用 Claude API（或项目配置的 LLM）生成文本分镜
- 输出格式固定，包含"创意要点"和"剧本内容"两部分
- 镜头数量由 AI 根据内容自动决定

## 与标准流程的区别

| 特性 | video-standard-create | video-quick-create |
|------|---------------------|-------------------|
| 交互 | 多轮交互确认 | 一次性生成 |
| 图片 | 多张候选让用户选 | 1张直接输出 |
| 分镜 | 图片故事板 | 文本分镜 |
| 速度 | 慢，但可精细调整 | 快 |
| 适用场景 | 需要精细控制 | 快速出片 |
