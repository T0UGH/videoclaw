# 分镜多类型支持设计文档

## 背景

当前 video-text-storyboard skill 只支持故事类分镜模板。用户需要产品展示类、角色动作类、风景旅拍类三种新模板，并支持用户自带图片或 AI 生成两种素材来源。

## 目标

- 新增三种分镜类型：产品展示、角色动作、风景旅拍
- 更新追问逻辑，先锁定类型再问类型专属问题
- 每种类型有独立的 references 文件和示例

## 文件结构

```
skills/video-text-storyboard/
├── SKILL.md                     # 更新追问逻辑 + 模板选择
└── references/
    ├── stories.md               # 已有，不动
    ├── product.md               # 新增：产品展示类
    ├── action.md                # 新增：角色动作类
    └── scenic.md                # 新增：风景旅拍类
```

## SKILL.md 追问逻辑

### 追问顺序

1. **视频类型**（第一个问，用 AskUserQuestion 多选一）
   - 故事类（情感短片、微电影）
   - 产品展示类（品牌广告、电商视频）
   - 角色动作类（武侠、舞蹈、格斗）
   - 风景旅拍类（自然风光、城市街拍）

2. **视频主题**：这个视频想讲什么 / 展示什么？

3. **素材来源**（用 AskUserQuestion 二选一）
   - 我有图片，直接上传
   - 帮我用 AI 生成素材图

4. **类型专属追问**（根据类型分支）

| 类型 | 追问内容 |
|------|---------|
| 故事类 | 角色外观描述、风格（搞笑/温馨/史诗感） |
| 产品展示类 | 产品名称和外观描述、品牌调性（高端/科技感/轻快时尚） |
| 角色动作类 | 角色外观描述、动作类型（武侠/舞蹈/格斗/特技） |
| 风景旅拍类 | 拍摄地点/场景描述、氛围（治愈系/史诗感/纪录片风格） |

5. **时长**：15秒 / 30秒

### 模板选择逻辑

```
类型 → 加载对应 references 文件
故事类     → references/stories.md
产品展示类 → references/product.md
角色动作类 → references/action.md
风景旅拍类 → references/scenic.md
```

## 各类型模板设计

### 通用主结构（所有类型一致）

```
【素材介绍】：
【@图1】<描述>
【@图2】<描述>

【场景描写】：
<整体背景描述>

【背景音】：
<背景音描述>

【分镜】：
【镜头标题】（X-Y秒）
画面：<描述>
音效：<音效>

【注意事项】：
- ...
```

### 产品展示类差异

**固定镜头节奏**：
1. 开场抓眼（悬念/特写）
2. 产品全景展示（环绕/推拉）
3. 产品细节特写（材质/工艺）
4. 使用场景（实际环境）
5. 品牌落版（slogan）

**注意事项专属字段**：
- 品牌调性：<高端/科技感/轻快时尚>
- 产品核心卖点：<需要重点展示的特性>

### 角色动作类差异

**固定镜头节奏**：
1. 角色亮相（定格/缓慢展示造型）
2. 动作起势（准备姿势）
3. 核心动作（打斗/舞蹈/特技）
4. 收尾定格（pose 定格）
5. 特效强化（氛围/画面落版）

**注意事项专属字段**：
- 动作连贯性：<关键动作衔接要求>
- 运镜要求：<跟随动作的镜头方式>

### 风景旅拍类差异

**素材介绍**：以场景图为主，角色可选。

**固定镜头节奏**：
1. 大景别建立（全貌）
2. 中景推进（引入人物或细节）
3. 多角度切换（环境不同面貌）
4. 特写细节（光影/纹理）
5. 意境落版（大景别或空镜）

**注意事项专属字段**：
- 光影要求：<时段/光线特征>
- 色调：<整体色彩倾向>

## 素材来源处理

### 用户自带图片
- 提示用户上传图片
- 直接在分镜中用 【@图N】 引用

### AI 生成素材图
- 根据用户描述，先调用 t2i 生成素材图
- 生成后再进入分镜生成流程
- 流程与 video-quick-create 中的 t2i 步骤一致

## 不涉及改动的内容

- `video-quick-create/SKILL.md`：本次不改，quick-create 走的是固定故事类流程
- `video-i2v-from-storyboard/SKILL.md`：不改，执行层无需感知分镜类型
- references/stories.md：不改
